|admin|
-----------------

``drf_admin`` 基于Django、Django REST framework（DRF）、Channels、Redis、Vue开发的，前后端分离后台管理系统，项目采用分模块开发方式， 权限控制采用 RBAC

开发初衷
-----------------

1. ``drf_admin`` 个人业余时间练习的项目
2. ``eladmin`` 感觉大佬项目写的很棒，于是参考大佬项目页面布局开发

主要应用技术
-----------------
1. Django
2. Django REST framework
3. Channels
4. Redis
5. Vue
6. Element UI
